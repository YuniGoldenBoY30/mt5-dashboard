# 💎 Frontend VIP: Sistema de Diseño Obsidian

El Dashboard de QuantFib VIP utiliza una interfaz de usuario de vanguardia diseñada para transmitir confianza, precisión y exclusividad.

## 1. Estética Obsidian VIP
Abandonamos el glassmorphism tradicional por una estética de **Terminal de Grado Institucional**:
- **Bento Grids:** Organización modular basada en tarjetas sólidas con jerarquía clara.
- **Deep Obsidian Palette:** Fondo `#050505` con contenedores `#0f0f0f`.
- **Institutional Gold:** Acento principal para elementos VIP y resultados de alta rentabilidad.
- **Cyber Cyan:** Acento técnico para indicadores de Inteligencia Artificial.

## 2. Puntos Técnicos
- **Framework:** React + TypeScript + Vite.
- **Styling:** Tailwind CSS con tokens personalizados (`obsidian-bg`, `obsidian-card`, etc.).
- **Real-Time:** Integración nativa con WebSockets para actualizaciones de milisegundos sin recarga de página.
- **Data Viz:** Gráficos de alta fidelidad con efectos de glow para tendencias de Equity y Drawdown.

## 3. Vistas Principales
- **Dashboard:** Resumen ejecutivo de todas las cuentas y alertas activas.
- **Team View:** Enfoque en métricas de rendimiento para gestores de fondos.
- **Dev View:** Acceso profundo a telemetría de IA, logs de auditoría y controles operacionales (Cierre de posiciones).
